# TypechoPlugins

这里放置的是Typecho插件，所有插件均遵循GPL-3.0 license。

## 插件列表

| 插件文件夹 | 插件名称 | 功能描述 |
|-----------|---------|---------|
| AMapPlugin | 高德地图插件 | 用于在文章中插入交互式高德地图 |
| AMapRoutePlugin | 高德地图路径规划插件 | 用于在文章中插入交互式高德地图路径规划 |
| AdminImagePreview | 后台图片预览插件 | 为Typecho后台附件管理页面和文章编辑页面添加图片预览功能 |
| AdminStyle | 后台样式修改插件 | 用于修改Typecho后台管理界面样式，支持多种预设主题和自定义CSS代码 |
| BilibiliPlayer | Bilibili播放器插件 | 将Bilibili默认播放器替换为HTML5移动端播放器，并提供更多自定义选项 |
| Panorama | 全景图插件 | 在文章中嵌入全景图，支持等距柱状投影和立方体贴图两种类型 |

如果觉得插件不错，请给个star，或者给予捐助，谢谢。
