# TypechoPlugins

这里放置的是目前我使用的Typecho插件。

## 插件列表

| 插件文件夹 | 插件名称 | 功能描述 |
|-----------|---------|---------|
| AISummary | AI摘要插件 | 使用AI为Typecho文章生成摘要，支持ChatGPT及相似API调用方式的AI模型 |
| Akismet | 反垃圾评论插件 | 使用Akismet服务过滤垃圾评论，保护博客免受垃圾信息干扰 |
| AMapPlugin | 高德地图插件 | 用于在文章中插入交互式高德地图 |
| AMapRoutePlugin | 高德地图路径规划插件 | 用于在文章中插入交互式高德地图路径规划 |
| AdminImagePreview | 后台图片预览插件 | 为Typecho后台附件管理页面和文章编辑页面添加图片预览功能 |
| AdminStyle | 后台样式修改插件 | 用于修改Typecho后台管理界面样式，支持多种预设主题和自定义CSS代码 |
| BilibiliPlayer | Bilibili播放器插件 | 将Bilibili默认播放器替换为HTML5移动端播放器，并提供更多自定义选项 |
| DouYinPlayer | 抖音播放器插件 | 通过抖音视频ID获取IFrame代码，实现抖音视频播放 |
| FlowChart | 流程图插件 | 在文章中渲染流程图，使用flowchart.js库将文本格式转换为可视化的SVG流程图 |
| Meting | 音乐播放器插件 | 在Typecho中使用APlayer播放在线音乐，支持多种音乐平台 |
| MusicPlayer | 音乐播放器插件 | 自动将网易云音乐和QQ音乐链接转换为可嵌入的播放器，支持歌曲、歌单和专辑 |
| HighlightJs | 代码高亮插件 | 为文章中的代码块提供语法高亮显示 |
| Mailer | 邮件通知插件 | 评论回复时发送邮件通知 |
| Panorama | 全景图插件 | 在文章中嵌入全景图，支持等距柱状投影和立方体贴图两种类型 |
| Sitemap | 站点地图插件 | 自动生成XML格式的站点地图，便于搜索引擎索引 |
| SwitchImges | 图片格式转换插件 | 上传图片时自动转换为WebP/AVIF格式，支持GD/ImageMagick/FFmpeg多种处理方式 |
| WordsCounter | 字数统计插件 | 统计文章和全站字数，支持中文字数统计 |

如果觉得插件不错，请给个star，或者给予捐助，谢谢。
